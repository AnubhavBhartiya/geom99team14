Intro

This was a Group Project to fulfil the requirements of the Course GEOM099 which required to plan out a problem statement and frame out a web GIS Solution so that WEB GIS Solutions can be used to get the output of the project.

In this project, we were expected to explore the available web technologies as a group. This was a five-week project where we had to figure out a problem statement and come up with a web solution for the same. We had to demonstrate how that particular web solution would be effective in solving that particular problem. This project also required us to maintain individual logs so that the work timeline could be monitored and noted

Problem Statement

Intially to start the project we had to identify a Problem Statement. A problem statement is basically a concise description of an issue to be adressed. The Problem statement had several parts to it. The very first part was to identify the Ideal followed by the reality of what is the condition which lead to the Consequences of what happens to the project and how would it put an impact. At last we had to write down a proposal of what output we were to provide by the end of the project.

Ideal

Using open source data from Ontario GeoHub, we want to utilize Historical Fish Stocking Data within Kawartha Highlands Provincial Park - from here we can navigate onto a Dashboard that allows the user to move around the map and see annual stocking numbers over time. This can also be used in a Storymap format that provides a descriptive historical background. Prioritize the Dashboard, Storymap if time allows.

Reality

Ontario currently runs on Fish ON-Line, a meteocre design that has gaps in data and manuevarability. This new interactive dashboard could potentially provide new opportunities as a template for other parks to do the same moving forward.

Consequences

The public may be unaware of the opportunities for outdoor recreation that Kawartha Highlands Provincal Park provides and may never see the parks beauty. Furthermore, there is no way to currently view historic stocking data for Kawartha Highlands Provincal Park.

Proposal

Present an interactive map that can be used to find fishing spots in Kawarth Highlands Park and easily view the stocking history. The story map will also highlight the history and popular attractions of the park.

Technology Used

ArcGIS Online, ArcGIS Dashboards, Story Map. Basically we used the ESRI Softwares to execute the Web Part of the project

Deliverables

For the final output of the project we plannes out two major deliverables which were a Dashboard marking the Fish Stocks in the lakes present in the Kawartha Highlands Provincial Park and a Story Map marking all the Fishing spots in the area and also a short story about the area and how it transformed through time.

Fish Stock Data Dashboard

This is Dashboard made to mark the Fish stocking done in each lake. It shows the latest stocking data of all the species of Fishes there in the lake and all the previous fish stocking data recorded is also updated on the dashboard. This dashboard has three major segments which are a map Highlighting the Kawartha Highland Provincial Park, Followed by a Pie Chart showing the share of a particular species of the fish present in the particular area. At last is the table which shows the species of the fish name of the lake and the quantity which was stocked in which particular lake. All of this is Geographically interlinked so when the area is zoomed in the map all the data on the Pie Chart and the Table automatically changes according to the area visible in the map

This Map was created by using the shape file of the area of Kawartha Highlands Provincial Park and all the data available on the Government websites regarding Fish Stocking.

Fishing Species story Map

This is a Story Map solution available by ESRI in which We can present a story line with Maps and Geospatial Data. For this we had hosted WebMaps on ArcGIS Online and then used that data and put it on the Story Timeline. So the Timeline present here in this storymaps shows how the park changed with time and how the boundary area of that park differed. Further that story map also shows the fishing points and more information related to the fishing spots present in the Kawartha Highland Provincial Park.
